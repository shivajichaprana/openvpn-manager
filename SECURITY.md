## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email: open a [GitHub Security Advisory](https://github.com/shivajichaprana/openvpn-manager/security/advisories/new) (private disclosure).

You will receive a response within 72 hours. If confirmed, a patch will be released within 7 days.
